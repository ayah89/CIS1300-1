steps:

  1. create an account with github
    • some of you may have one already
    • i dont care what your account name is, but make sure it SFW
  2. create a new repository
    • call it "CIS 1300"
    • you don't have to make it the sameas your username (like the guide instructions)
  3. Make the repo a GitHub Pages enabled repo
    • repo_name > settings > options > scroll down to "Github pages"
    • select either "master branch" or "master branch / docs folder"
      • master branch = all files are now pages
      • master branch / docs folder = you have a docs folder for pages, everything else is a normal repo
  4. Take note of the URL GitHub gives you, this is your web address
  5. Make an index.html page, this will be your home page
  6. You will put links on this page later for all your assignments
  7. All your assignments will be turned in as links to the LIVE files online
  8. GitHub keeps track of the upload date for all your files so I will know if you finished them after the due date
    • you can re-upload things many times
    • you can also view things locally before you upload them
  9. You will NEED to learn how to use GitHub, just a little, enough to commit new files
